# ogbl-collab

### nodeidx2authorid.csv.gz

Mapping from node index into the unique author id used in Microsoft Academic Graph. 
The actual paper can be found by https://academic.microsoft.com/author/1878631932, where you need to replace the number (in this case, 1878631932) with the specified author id.
